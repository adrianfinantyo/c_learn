// MENGAPA DILAKUKAN REHASHING? //
    Rehashing dilakukan untuk mencegah terjadinya collision
    dengan cara memperbesar ukuran hash table sehingga setiap
    index data memiliki value yang unik.

// SEBUTKAN LANGKAH-LANGKAH MELAKUKAN REHASHING SECARA
   SINGKAT, PADAT, DAN JELAS! //

    atau tidak. Jika ya, maka harus diperbesar dengan
    memperbesar 2x lipat dan lebih besar dari bilangan prima terdekat.
